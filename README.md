# Aura Chat

A modern AI chat interface built with React, Vite, and Tailwind CSS.

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
